# Dynamic UI – Jewellery Template Builder (React + Konva)

A focused template editor for jewellery creatives using **React** and **Canvas** via **react-konva**.

## What it supports

- Dynamic text fields for:
  - Date
  - Gold rates (1GM 22K, 8GM 22K, 1GM 18K)
  - Phone number
  - Address + secondary footer line
  - Message text
  - Brand name
- Customer uploads:
  - Background image
  - Logo image
- Drag repositioning for all major blocks (logo/title/date card/rate card/message/footer)

## Run locally

```bash
npm install
npm run dev
```
